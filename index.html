ℹ️ 导入格式说明

TXT 格式：每行一条反馈，系统会自动识别分类和优先级

CSV 格式：表头包含 title,category,priority,description 字段

JSON 格式：数组对象，每个对象包含 title, category, priority, description 字段

  • 需求建议(feature)：希望、建议、增加、添加、支持、功能、优化、改进
